# Tearsheet AI

**Stock analysis tool with a 6-pillar scoring system, intrinsic value models, and AI-generated narratives.**

Pick a company → get a tear sheet with:
- Quant scores (moat, financial health, growth, valuation, technical, governance)
- Intrinsic value range (Ben Graham, DCF, Dhandho, Expected Returns)
- Red flags checklist
- AI narrative (thesis, catalysts, risks)

**Deployed version:** See DEPLOY.md for how to get a live link (Streamlit Cloud, free)

**No cost, no API key needed (uses free copy-paste-into-Claude.ai mode by default).**
